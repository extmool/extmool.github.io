当你的同学没有锁电脑的时候……

以下行为可能会招来一顿毒打，所有情况均与本站无关[大雾]

以下命令均在 Windows Server 2008 R2 测试通过，其他系统请自测。

以下所有命令需要写成 bat 文件运行

轻微毒瘤

1. 蓝屏死机

2. 更改文件后缀名

中度毒瘤

3. 删除system32

4. 使PC永久崩溃

5. 删除所有的注册表

重度毒瘤

6. 永久禁用网络

7. 一直按回车

8. 启动电脑后自动关机

极致毒瘤

9. 开启CD刻录机蜂鸣器

注意，这些命令可能会使您的耳朵崩溃
